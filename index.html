2. Infrastructure Provisioning

We offer two delivery models for the SQL environment. Both utilize Infrastructure as Code (IaC) for stability and transparency.

Option 1: Client-Owned Cloud

We build on your internal accounts.

Ownership

We deploy the entire architecture directly into your existing **GCP or Azure** account. This is ideal if you want full control over the primary billing and infrastructure credentials from day one.

  • ✓ Full hand-off of IaC (Infrastructure as Code) scripts.
  • ✓ Setup, troubleshooting & scaling documentation included.
  • ✓ Direct integration with your existing security policies.

Fee Structure

One-time Implementation Fee

Option 2: Mason-Managed

Build & Host & Manage Solution

Value-Add

Our most robust option. All resources are primary-provisioned on **GCP**, but we utilize IaC to provide **Failover Redundancy** across Azure and AWS for an "always-on" experience.

  • ✓ Multi-Cloud Redundancy: Automatic provisioning across Azure/AWS if GCP fails.
  • ✓ Maintenance: Ongoing PowerBI dashboard maintenance and updates included.
  • ✓ Monitoring: Continuous schema monitoring and data sync health checks.
